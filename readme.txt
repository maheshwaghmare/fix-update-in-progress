== Description ==

Zero configuration. One click fix "Another update is currently in progress." which occurs while updating the WordPress core or auto updating the thems & plugins.

== Installation ==

1. Install the <code>Fix Update in Process</code> plugin either via the WordPress plugin directory, or by uploading the files to your server at <code>wp-content/plugins</code>.
2. Goto `Tools > Fix Update in Process`
3. Click on `Release All Locks`.
